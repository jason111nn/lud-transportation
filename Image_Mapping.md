# LHD 海報圖片對應清單 (Image Mapping)

本清單列出 `PdfWorkspace.vue` 中所有 `<img>` 占位符對應的路徑與應截取的報告內容。
請將您的報告截圖或照片更名為以下檔名，並放入 `public/img/` 資料夾內。網頁渲染時將會自動讀取並顯示在海報的正確位置上。

| 程式碼中的 `src` 路徑 | 對應海報位置 | 描述與建議 |
| :--- | :--- | :--- |
| `/img/behavior_analysis.png` | **Panel 1: 製作動機** | 圖表放置關於停等紅燈低頭族的數據統計或行為分析圖表。 |
| `/img/system_arch.png` | **Panel 2: 系統摘要** | 包含 ESP32、BLE 廣播與手機 App 接收的系統架構方塊圖。 |
| `/img/circuit_design.png` | **Panel 2: 製作流程(硬體)** | KiCad 電路圖、草稿或相關硬體設計圖面。 |
| `/img/arduino_code.png` | **Panel 2: 製作流程(軟體)** | Arduino IDE 程式碼截圖。 |
| `/img/breadboard.png` | **Panel 2: 製作流程(硬體)** | 麵包板與實體接線測試的照片。 |
| `/img/packet_5byte.png` | **Panel 3: 製作成果** | 關於 5-Byte BLE 廣播封包結構圖解或解析示意圖。 |
| `/img/app_mockup.png` | **Panel 3: 製作成果** | 手機 App 顯示「要綠燈了」的通知介面或模擬畫面截圖。 |
| `/img/android_dev.png` | **Panel 3: 製作流程(軟體)** | Android Studio 或 Jetpack Compose 開發介面截圖。 |
| `/img/3d_print.png` | **Panel 3: 製作流程(硬體)** | 3D 列印號誌外殼的設計圖或正在列印中的照片。 |
| `/img/final_model.png` | **Panel 3: 流程終點** | 十字路口模型與硬體安裝完工的最終實拍照。 |
| `/img/data_structure.png` | **Panel 4: 未來展望** | 關於系統封包擴充或大數據傳輸架構的升級圖解。 |
| `/img/v2i_concept.png` | **Panel 4: 未來展望** | 關於 V2I（車路協同）或智慧交通連結的概念示意圖。 |
| `/img/og-image.png` | **Social Media Sharing** | 網站社群分享卡片圖片（建議 1200x630）。 |
